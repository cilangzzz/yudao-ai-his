## 1. 功能概述

### 1.1 功能定位

病历质控模块是电子病历子系统的质量管理核心，负责对病历进行时限质控和完整性质控，自动检测病历书写时限是否达标、必填项是否完整，提供质控评分和整改通知，支持质控统计分析。系统实现质控覆盖率100%，确保病历质量符合电子病历应用水平分级评价标准。

### 1.2 业务目标

| 目标类型 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 合规目标 | 实现病历时限质控和完整性质控 | 质控覆盖率100% |
| 质量目标 | 提高病历书写及时性和完整性 | 平均质控得分>=90分 |
| 效率目标 | 自动化质控检查，减少人工抽查 | 自动质控比例>=95% |
| 管理目标 | 及时发现问题病历并督促整改 | 整改及时率>=90% |
| 智能目标 | 引入AI辅助病历质控 | AI质控准确率>=85% |

## 4. 交互流程

### 4.1 病历质控流程

```
病历审签完成
    │
    ↓
系统自动触发质控检查
    │
    ├── 时限质控检查
    │   ├── 入院记录24h内完成?
    │   │   ├── 是 → 不扣分
    │   │   └── 否 → 计算超时时间，扣分
    │   ├── 首次病程8h内完成?
    │   │   ├── 是 → 不扣分
    │   │   └── 否 → 计算超时时间，扣分
    │   ├── 日常病程每日记录?
    │   │   ├── 是 → 不扣分
    │   │   └── 否 → 缺一日扣5分
    │   ├── 出院记录48h内完成?
    │   │   ├── 是 → 不扣分
    │   │   └── 否 → 计算超时时间，扣分
    │   └── 上级医师查房记录频率检查
    │
    ├── 完整性质控检查
    │   ├── 主诉>=10字符?
    │   │   ├── 是 → 不扣分
    │   │   └── 否 → 扣10分
    │   ├── 现病史>=100字符?
    │   │   ├── 是 → 不扣分
    │   │   └── 否 → 扣20分
    │   ├── 既往史已填写?
    │   │   ├── 是 → 不扣分
    │   │   └── 否 → 扣10分
    │   ├── 诊断关联ICD-10?
    │   │   ├── 是 → 不扣分
    │   │   └── 否 → 扣15分
    │   └── 其他必填项检查...
    │
    ↓
计算质控综合得分
    │
    ↓
判定质控等级
    │
    ├── 得分>=90分 → 合格，记录质控结果
    │
    └── 得分<90分 → 不合格
        │
        ↓
    生成整改通知
        │
        ↓
    发送通知给责任医生
        │
        ↓
    医生查看整改要求
        │
        ↓
    医生修改病历
        │
        ↓
    确认整改完成 → 重新质控
```

### 4.2 整改通知流程

```
质控不合格
    │
    ↓
质控医师选择病历发送整改通知
    │
    ├── 填写整改期限（默认3天）
    ├── 系统自动生成整改要求
    └── 发送通知到责任医生
    │
    ↓
责任医生收到通知（系统消息/短信）
    │
    ↓
医生查看整改详情
    │
    ├── 查看扣分明细
    ├── 查看整改要求
    └── 查看整改期限
    │
    ↓
医生修改病历
    │
    ↓
医生点击"整改完成"
    │
    ↓
系统重新执行质控
    │
    ├── 质控通过 → 整改状态变更为"已确认"
    │
    └── 质控仍不合格 → 提示继续整改
```

### 5.2 质控等级定义

| 等级 | 得分范围 | 说明 |

## 6. 业务规则

### 6.1 时限质控规则

| 文书类型 | 完成时限 | 质控规则 | 扣分标准 |
|----------|----------|----------|----------|
| 入院记录 | 入院后24小时内 | 超时未完成扣分 | 超时扣5分/小时，上限30分 |
| 首次病程记录 | 入院后8小时内 | 超时未完成扣分 | 超时扣5分/小时，上限20分 |
| 日常病程记录 | 每日至少1次 | 未按时书写扣分 | 缺一日扣5分 |
| 上级医师查房记录 | 主治医师每周2次，主任每周1次 | 未按时查房扣分 | 缺一次扣5分 |
| 出院记录 | 出院后48小时内 | 超时未完成扣分 | 超时扣5分/小时，上限20分 |
| 死亡记录 | 死亡后24小时内 | 超时未完成扣分 | 超时扣10分/小时，上限30分 |
| 知情同意书 | 操作前签署 | 未签署扣分 | 未签署扣20分 |

### 6.2 完整性质控规则

| 检查项 | 文书类型 | 质控规则 | 扣分标准 |
|----------|----------|----------|----------|
| 主诉 | 入院记录 | 必填，>=10字符 | 缺失扣10分，不足扣5分 |
| 现病史 | 入院记录 | 必填，>=100字符 | 缺失扣20分，不足扣10分 |
| 既往史 | 入院记录 | 必填 | 缺失扣10分 |
| 体格检查 | 入院记录 | 必填 | 缺失扣10分 |
| 初步诊断 | 入院记录 | 必填，关联ICD-10 | 缺失扣15分 |
| 病例特点 | 首次病程记录 | 必填，>=50字符 | 缺失扣10分，不足扣5分 |
| 诊断依据 | 首次病程记录 | 必填 | 缺失扣10分 |
| 鉴别诊断 | 首次病程记录 | 必填 | 缺失扣10分 |
| 诊疗计划 | 首次病程记录 | 必填 | 缺失扣10分 |
| 出院医嘱 | 出院记录 | 必填 | 缺失扣10分 |

### 6.3 质控触发时机

| 触发场景 | 触发时机 | 质控类型 |

| 病历提交审签 | 提交前预检 | 完整性质控 |
| 定时任务 | 每日凌晨执行 | 时限质控 |
| 手动触发 | 质控医师操作 | 综合质控 |

### 6.4 整改时限规则

| 规则项 | 规则内容 |
