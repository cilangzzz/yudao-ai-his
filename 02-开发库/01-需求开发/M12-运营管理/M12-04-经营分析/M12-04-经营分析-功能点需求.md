## 1. 功能概述

### 1.1 功能定位

经营分析模块是运营管理子系统的决策支持模块，面向医院管理层提供收入分析、成本分析等经营决策支持功能。系统整合门诊、住院、财务等业务数据，构建经营数据分析模型，帮助医院管理层了解经营状况、发现经营问题、制定经营策略。

### 1.2 业务目标

| 目标类型 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 决策目标 | 为经营决策提供数据支撑 | 关键指标实时更新 |
| 效率目标 | 提升经营分析效率 | 数据查询响应时间≤5秒 |
| 管理目标 | 实现收入成本透明化管理 | 经营数据自动采集率≥95% |
| 合规目标 | 数据统计符合卫健委要求 | 报表格式符合规范 |

### 1.3 功能范围

```
M12-04 经营分析
├── 收入分析
│   ├── 收入构成分析（门诊收入、住院收入、药品收入、其他收入）
│   ├── 收入趋势分析（月度趋势、同比增长、环比增长）
│   ├── 科室收入对比（科室收入排名、收入增长率）
│   ├── 收入明细穿透（门诊收入明细、住院收入明细、药品收入明细）
│   └── 收入分析报告导出
└── 成本分析
    ├── 成本构成分析（人力成本、药品成本、设备折旧、管理费用）
    ├── 科室成本分析（科室成本TOP5、成本占比）
    ├── 收支结余分析（总收入与总成本对比、成本收益率）
    ├── 成本趋势分析（月度成本趋势）
    └── 成本分析报告导出
```

### 1.4 用户角色

| 角色 | 主要职责 | 使用功能 |
|------|----------|----------|
| 院长/副院长 | 全院经营决策 | 收入分析、成本分析、收支结余分析 |
| 财务人员 | 财务数据分析 | 收入分析、成本分析、报表导出 |
| 科室主任 | 科室经营分析 | 本科室收入成本分析 |


```gherkin
Feature: 收入分析

  Scenario: 查看收入构成分析
    Given 院长已登录运营管理系统
    When 院长进入收入分析页面
    Then 系统显示收入构成饼图
    And 显示门诊收入占比和金额
    And 显示住院收入占比和金额
    And 显示药品收入占比和金额
    And 显示其他收入占比和金额

  Scenario: 查看收入趋势分析
    Given 院长在收入分析页面
    When 院长选择分析周期"近12个月"
    Then 系统显示收入趋势折线图
    And 显示同比增长率
    And 标注收入峰值和谷值月份

  Scenario: 查看科室收入对比
    Given 院长在收入分析页面
    When 院长选择"科室对比分析"
    Then 系统显示各科室收入对比图表
    And 显示科室收入排名
    And 显示科室收入增长率

  Scenario: 查看收入明细穿透
    Given 院长在收入分析页面
    When 院长点击某科室收入数据
    Then 系统显示该科室收入明细
    And 显示门诊收入明细
    And 显示住院收入明细
    And 显示药品收入明细

  Scenario Boundary: 无收入数据时段
    Given 选定时间段无收入数据
    When 院长查看收入分析
    Then 系统显示"暂无数据"提示
    And 图表显示空状态
```

### 7.2 关键业务规则详解

#### BR-OPS-018: 收入构成占比计算

收入构成占比计算规则：各项收入占比 = 各项收入金额 / 总收入 × 100%

**计算维度**:
- 门诊收入：门诊挂号费 + 门诊诊查费 + 门诊检查费 + 门诊治疗费
- 住院收入：住院床位费 + 住院诊查费 + 住院治疗费 + 住院护理费
- 药品收入：门诊药品收入 + 住院药品收入
- 其他收入：检查报告打印费 + 证明文件费 + 其他服务费

**触发时机**: 收入分析查询时

**计算逻辑**:
```
总收入 = 门诊收入 + 住院收入 + 药品收入 + 其他收入

各项占比：
门诊收入占比 = 门诊收入 / 总收入 × 100%
住院收入占比 = 住院收入 / 总收入 × 100%
药品收入占比 = 药品收入 / 总收入 × 100%
其他收入占比 = 其他收入 / 总收入 × 100%
```

#### BR-OPS-023: 成本收益率计算

成本收益率计算规则：成本收益率 = 总收入 / 总成本 × 100%

**成本分类**:
- 人力成本：人员工资 + 社保公积金 + 培训费用
- 药品成本：药品采购成本 + 药品损耗
- 设备折旧：医疗设备折旧 + 办公设备折旧
- 管理费用：水电费 + 办公费 + 差旅费 + 其他管理费用

**计算逻辑**:
```
总成本 = 人力成本 + 药品成本 + 设备折旧 + 管理费用
收支结余 = 总收入 - 总成本
成本收益率 = 总收入 / 总成本 × 100%

判断标准：
成本收益率 > 100%：盈利状态
成本收益率 = 100%：收支平衡
成本收益率 < 100%：亏损状态
```

#### BR-OPS-014: 经营分析数据权限

经营分析数据为敏感数据，仅特定角色可查看：

**权限矩阵**:
| 角色 | 收入分析 | 成本分析 | 收支结余 | 导出报告 |
|------|----------|----------|----------|----------|
| 院长/副院长 | 全院数据 | 全院数据 | 全院数据 | 允许 |
| 财务负责人 | 全院数据 | 全院数据 | 全院数据 | 允许 |
| 财务人员 | 全院数据 | 部分数据 | 部分数据 | 需审批 |
| 科室主任 | 本科室 | 本科室 | 无权限 | 需审批 |
| 其他人员 | 需授权 | 需授权 | 需授权 | 需审批 |
