## 1. 功能概述

### 1.1 功能定位

租户管理模块实现SaaS多租户架构的核心管理能力。支持租户创建、租户套餐、租户数据隔离、租户状态管理。通过租户隔离，实现一套系统服务多个医疗机构（医院/诊所/社区卫生中心），每个租户数据完全隔离。

### 1.2 业务目标

| 目标类型 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 数据隔离目标 | 租户间数据完全隔离 | 数据隔离率100% |
| 运营目标 | 支持多机构独立运营 | 支持租户数≥100 |
| 管理目标 | 租户套餐灵活配置 | 套餐可定制 |

### 2.4 租户数据隔离机制

| 隔离方式 | 实现方法 | 适用场景 |
|----------|----------|----------|
| 字段隔离 | 表增加tenant_id字段 | 大部分业务表 |
| 数据库隔离 | 独立数据库 | 高安全要求租户 |
| Schema隔离 | 独立Schema | 中等安全要求 |

**字段隔离实现**:
```sql
-- 所有业务表增加tenant_id字段
CREATE TABLE his_patient (
  id BIGINT PRIMARY KEY,
  tenant_id BIGINT NOT NULL,  -- 租户ID
  name VARCHAR(50),
  id_card VARCHAR(18),
  ...
);

-- 自动追加租户条件
SELECT * FROM his_patient WHERE tenant_id = #{tenantId}
```
