| 属性 | 内容 |
|------|------|
| 规则编号 | BR-SYS-001 |
| 规则名称 | 用户名格式校验 |
| 规则类型 | 校验规则 |

| 优先级 | P0 |

**校验逻辑伪代码**：
```
FUNCTION validateUsername(username)
  IF length(username) < 3 OR length(username) > 20 THEN
    RETURN ERROR("用户名长度3-20位")
  END IF
  IF NOT match(username, "^[a-zA-Z][a-zA-Z0-9_]{2,19}$") THEN
    RETURN ERROR("用户名需以字母开头，仅支持字母、数字、下划线")
  END IF
  RETURN SUCCESS
END FUNCTION
```

#### BR-SYS-004: 密码复杂度校验

| 属性 | 内容 |
|------|------|
| 规则编号 | BR-SYS-004 |
| 规则名称 | 密码复杂度校验 |
| 规则类型 | 校验规则 |
| 适用对象 | sys_user.password |
| 规则描述 | 密码必须≥8位，包含大写字母、小写字母、数字、特殊字符中至少3类 |
| 触发时机 | 创建用户/修改密码时 |
| 校验逻辑 | `length >= 8 AND (has_upper + has_lower + has_digit + has_special) >= 3` |
| 失败处理 | 返回错误"密码不满足复杂度要求：需≥8位，包含大小写字母、数字、特殊字符至少3类" |
| 优先级 | P0 |

**校验逻辑伪代码**：
```
FUNCTION validatePassword(password)
  IF length(password) < 8 THEN
    RETURN ERROR("密码长度至少8位")
  END IF
  
  has_upper = match(password, "[A-Z]")
  has_lower = match(password, "[a-z]")
  has_digit = match(password, "[0-9]")
  has_special = match(password, "[!@#$%^&*()_+\\-=\\[\\]{};':\"\\\\|,.<>\\/?]")
  
  type_count = has_upper + has_lower + has_digit + has_special
  IF type_count < 3 THEN
    RETURN ERROR("密码需包含大写字母、小写字母、数字、特殊字符至少3类")
  END IF
  
  RETURN SUCCESS
END FUNCTION
```

| 校验逻辑 | 见伪代码 |

| 优先级 | P0 |

**校验逻辑伪代码**：
```
FUNCTION checkLoginLock(user)
  # 检查是否锁定中
  IF user.status = 'locked' THEN
    IF now() - user.lock_time < 30 minutes THEN
      remaining = 30 minutes - (now() - user.lock_time)
      RETURN ERROR("账户已锁定，请等待" + remaining + "分钟后再尝试")
    ELSE
      # 自动解锁
      user.status = 'normal'
      user.fail_count = 0
      user.lock_time = NULL
      UPDATE user
    END IF
  END IF
  
  RETURN SUCCESS
END FUNCTION

FUNCTION handleLoginFail(user)
  user.fail_count = user.fail_count + 1
  
  IF user.fail_count >= 5 THEN
    user.status = 'locked'
    user.lock_time = now()
    UPDATE user
    RETURN ERROR("登录失败次数过多，账户已锁定30分钟")
  END IF
  
  UPDATE user
  remaining = 5 - user.fail_count
  RETURN ERROR("用户名或密码错误，还剩" + remaining + "次机会")
END FUNCTION
```

#### BR-SYS-014: 数据权限过滤

| 属性 | 内容 |
|------|------|
| 规则编号 | BR-SYS-014 |
| 规则名称 | 数据权限过滤规则 |
| 规则类型 | 权限规则 |
| 适用对象 | 数据查询SQL |
| 规则描述 | 根据角色的数据权限范围过滤查询数据 |
| 触发时机 | 数据查询时 |
| 校验逻辑 | 见下表 |
| 优先级 | P0 |

**数据权限范围SQL条件**：

| 数据范围 | 标识值 | SQL条件 |
|----------|--------|---------|
| 全部数据 | 1 | 无限制 |
| 自定义数据 | 2 | dept_id IN (自定义科室列表) |
| 本部门数据 | 3 | dept_id = 当前用户科室 |
| 本部门及下级 | 4 | dept_id IN (当前科室及下级科室) |
| 仅本人数据 | 5 | create_by = 当前用户 |

**校验逻辑伪代码**：
```
FUNCTION applyDataScope(sql, user)
  roles = getUserRoles(user.user_id)
  data_scope = MAX(roles.data_scope)  # 取最大权限
  
  SWITCH data_scope
    CASE 1:  # 全部数据
      RETURN sql
    CASE 2:  # 自定义数据
      dept_ids = getCustomDeptIds(roles)
      RETURN sql + " AND dept_id IN (" + dept_ids + ")"
    CASE 3:  # 本部门数据
      RETURN sql + " AND dept_id = " + user.dept_id
    CASE 4:  # 本部门及下级
      dept_ids = getDeptAndChildren(user.dept_id)
      RETURN sql + " AND dept_id IN (" + dept_ids + ")"
    CASE 5:  # 仅本人数据
      RETURN sql + " AND create_by = '" + user.username + "'"
  END SWITCH
END FUNCTION
```

**生成逻辑**：
```
FUNCTION generateInitPassword()
  chars_upper = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
  chars_lower = "abcdefghijklmnopqrstuvwxyz"
  chars_digit = "0123456789"
  chars_special = "!@#$%^&*"
  
  password = ""
  password += random_char(chars_upper)
  password += random_char(chars_lower)
  password += random_char(chars_digit)
  password += random_char(chars_special)
  
  all_chars = chars_upper + chars_lower + chars_digit + chars_special
  FOR i = 4 TO 7
    password += random_char(all_chars)
  END FOR
  
  RETURN shuffle(password)
END FUNCTION
```
