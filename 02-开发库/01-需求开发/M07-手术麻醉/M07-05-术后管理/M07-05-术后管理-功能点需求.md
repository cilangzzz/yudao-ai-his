## 1. 功能概述

### 1.1 功能定位

术后管理是手术麻醉子系统的闭环模块，负责术后患者麻醉复苏期观察记录和术后随访管理。系统覆盖从手术完成到患者安全返回病房的复苏观察全过程，以及术后第1天、第3天、第7天的随访追踪，确保患者安全度过围术期。

### 1.2 业务目标

| 目标类型 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 安全目标 | 确保患者安全度过麻醉复苏期 | 复苏期并发症发生率≤1% |
| 质量目标 | 规范术后随访流程 | 术后随访完成率≥95% |
| 时效目标 | 及时发现术后并发症 | 异常情况上报及时率100% |
| 合规目标 | 符合麻醉质控要求 | Steward评分记录完整率100% |

## 3. 用户故事

### 3.1 US-M07-016 术后观察

**故事卡片**
```
作为 麻醉医生
我想要 完成术后患者麻醉复苏期观察记录
以便于 确保患者安全度过麻醉复苏期
```

**优先级**: P0（MVP必需）

**故事点**: 5

**验收标准 (Gherkin)**
```gherkin
Feature: 术后观察

  Scenario: 术后观察记录
    Given 患者手术完成进入复苏室
    When 麻醉医生记录观察内容
    Then 记录意识状态（Steward评分）
    And 记录呼吸功能
    And 记录循环功能
    And 记录疼痛评分
    And 记录恶心呕吐情况

  Scenario: Steward评分
    Given 悠者正在复苏
    When 医生进行Steward评分
    Then 评分内容包括：清醒程度、呼吸道通畅程度、肢体活动
    And 计算总分

  Scenario: 达到出室标准
    Given 悠者复苏观察≥30分钟（全身麻醉）
    And Steward评分≥4分
    When 医生确认出室
    Then 记录出室时间和去向
    And 状态变更为"可出室"

  Scenario: 未达标准继续观察
    Given 悠者Steward评分<4分
    When 医生尝试确认出室
    Then 系统提示"未达到出室标准"
    And 需继续观察

  Scenario Boundary: 观察时长要求
    Given 全身麻醉悠者
    When 观察时长<30分钟
    Then 系统提示"全身麻醉患者观察时长需≥30分钟"
```

### 3.2 US-M07-017 术后随访

**故事卡片**
```
作为 麻醉医生
我想要 完成术后患者随访记录
以便于 追踪术后恢复情况和麻醉相关并发症
```

**优先级**: P1（重要）

**故事点**: 5

**验收标准 (Gherkin)**
```gherkin
Feature: 术后随访

  Scenario: 术后第1天随访
    Given 手术后第1天
    When 麻醉医生进行随访
    Then 记录术后恢复情况
    And 记录伤口愈合情况
    And 记录并发症情况
    And 记录麻醉相关并发症

  Scenario: 随访提醒
    Given 有术后随访任务
    When 到达随访时间
    Then 系统发送随访提醒通知

  Scenario: 发现异常情况
    Given 随访中发现并发症
    When 医生标记异常情况
    Then 系统记录异常
    And 通知病房医生
    And 需要上报的自动上报

  Scenario: 完成随访
    Given 完成术后第1、3、7天随访
    When 医生确认随访完成
    Then 随访状态变更为"已完成"

  Scenario Exception: 悠者已出院
    Given 术后随访未完成
    And 悠者已出院
    When 医生标记"患者已出院"
    Then 系统提示转为门诊随访
    And 发送随访提醒给悠者
```

### 7.2 关键业务规则详解

#### BR-SUR-012: 出室标准校验

出室标准校验规则适用于术后观察(postop_observation)，全身麻醉患者复苏观察>=30分钟，Steward评分>=4分方可出室。

**触发时机**: 确认出室时

**校验逻辑**:
```
// 伪代码
observation = SELECT * FROM postop_observation WHERE surgery_id = :surgery_id

// 观察时长校验
IF surgery.anesthesia_type = 'GENERAL' THEN
    observation_minutes = MINUTE(NOW() - observation.start_time)
    IF observation_minutes < 30 THEN
        RETURN error("全身麻醉患者观察时长需>=30分钟，当前" + observation_minutes + "分钟")
    END IF
END IF

// Steward评分校验
IF observation.steward_score < 4 THEN
    RETURN error("Steward评分<4分，未达到出室标准")
END IF

RETURN success()
```

**失败处理**: 返回错误提示"未达到出室标准"

#### BR-SUR-023: 术后随访流程

术后随访流程规则适用于术后随访(surgery_followup)，术后随访按时间节点执行：第1天、第3天、第7天；发现异常需上报。

**触发时机**: 定时任务触发

**执行逻辑**:
```
// 伪代码
followup_schedule = [
    {'day': 1, 'required': true},
    {'day': 3, 'required': true},
    {'day': 7, 'required': true}
]

surgery_date = surgery.end_time

FOR each item IN followup_schedule DO
    expected_date = surgery_date + item.day DAYS
    
    IF CURRENT_DATE >= expected_date THEN
        followup = SELECT * FROM surgery_followup 
                   WHERE surgery_id = :surgery_id AND followup_day = item.day
        
        IF followup IS NULL THEN
            SEND_reminder(doctor, "患者术后第" + item.day + "天随访待完成")
        END IF
    END IF
END FOR

RETURN success()
```

#### BR-SUR-039: Steward评分计算

Steward评分计算规则适用于术后观察(postop_observation)，Steward评分=清醒程度分+呼吸道通畅分+肢体活动分；总分>=4分可出室。

**触发时机**: 进行Steward评分时

**计算逻辑**:
```
// 伪代码
// 清醒程度：0-2分
// 0=完全清醒，1=呼唤可醒，2=对呼唤无反应

// 呼吸道通畅：0-2分
// 0=可按医嘱咳嗽，1=可自主维持，2=需支持维持

// 肢体活动：0-2分
// 0=肢体有意识活动，1=肢体无意识活动，2=肢体无活动

consciousness_score = :consciousness  // 0-2
airway_score = :airway               // 0-2
activity_score = :activity           // 0-2

steward_score = consciousness_score + airway_score + activity_score

observation.steward_score = steward_score
observation.steward_detail = {
    'consciousness': consciousness_score,
    'airway': airway_score,
    'activity': activity_score
}

IF steward_score >= 4 THEN
    observation.can_discharge = true
ELSE
    observation.can_discharge = false
END IF

RETURN steward_score
```
