## 1. 功能概述

### 1.1 功能定位

门诊药房管理模块实现处方接收、审核、调配、发药的全流程管理。支持麻精药品专项管理，实现"五专管理"（专人、专柜、专锁、专账、专方），确保麻精药品合规使用。系统自动扣减库存，打印用药指导单，实现药房业务的信息化和规范化管理。

### 1.2 业务目标

| 目标类型 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 安全目标 | 处方审核覆盖率100% | 处方审核通过率≥95% |
| 合规目标 | 麻精药品五专管理 | 麻精药品账实相符率100% |
| 效率目标 | 提高发药效率 | 平均发药时间≤5分钟 |
| 准确目标 | 确保发药准确 | 发药差错率≤0.1% |

## 3. 用户故事

### US-M01-04-001 处方接收

**故事卡片**
```
作为 药剂师
我想要 查看待发药处方列表
以便于 了解待处理处方情况
```

**优先级**: P0（MVP必需）

**故事点**: 2

**验收标准 (Gherkin)**
```gherkin
Feature: 处方接收

  Scenario: 查看待发药处方列表
    Given 药剂师已登录药房系统
    And 有待发药处方
    When 药剂师进入处方发药页面
    Then 系统显示待发药处方列表
    And 列表按缴费时间排序
    And 显示处方编号、患者姓名、科室

  Scenario: 查看处方详情
    Given 药剂师选择某处方
    When 药剂师点击"查看详情"
    Then 系统显示完整处方信息
    And 显示患者信息、诊断、药品明细

  Scenario: 患者未缴费提示
    Given 处方已开立但患者未缴费
    When 药剂师查看处方
    Then 系统显示"患者未缴费"状态
    And 药剂师无法进行发药操作
```

### 5.2 麻精药品管理规则详解

#### BR-OP-040: 麻精药品五专管理

麻醉药品必须"五专"管理：专人、专柜、专锁、专账、专方，发药时需双人核验签字。

**五专管理要求**:

| 专 | 要求 |
|----|------|
| 专人 | 指定专人负责麻精药品管理 |
| 专柜 | 使用专用保险柜存放 |
| 专锁 | 双人双锁管理 |
| 专账 | 建立专用账册，日清日结 |
| 专方 | 使用专用红色处方 |

**双人核验规则**:
- 发药药师和核验药师必须为不同人员
- 两人均需输入密码验证身份
- 核验通过后方可发药

#### BR-OP-011: 麻精药品专项校验

麻精药品处方必须符合"红处方"要求：开方医生有处方权、单次发药量不超过规定限量、需双人核验。

**限量标准（示例）**:

| 药品类型 | 门诊患者 | 癌症患者 |
|----------|----------|----------|
| 麻醉药品注射剂 | 1次常用量 | 3日常用量 |
| 麻醉药品片剂 | 3日常用量 | 7日常用量 |
| 第一类精神药品 | 3日常用量 | 7日常用量 |
