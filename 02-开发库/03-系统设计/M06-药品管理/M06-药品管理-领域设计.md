## 1. 界限上下文

### 1.1 药品上下文 (MedicationContext)

| 属性 | 值 |
|------|------|
| 上下文编号 | BC-003 |
| 英文标识 | MedicationContext |
| 边界范围 | 药品管理域 |
| 核心职责 | 管理药品目录、库存、处方审核、调配发药 |
| 对应模块 | M06-药品管理 |

```
┌─────────────────────────────────────────────────────────────────┐
│                      MedicationContext                          │
├─────────────────────────────────────────────────────────────────┤
│ 聚合根:                                                         │

│   - PrescriptionReview (处方审核)                               │
├─────────────────────────────────────────────────────────────────┤
│ 领域服务:                                                       │
│   - DrugCatalogService (药品目录服务)                           │
│   - InventoryService (库存管理服务)                             │
│   - PrescriptionReviewService (处方审核服务)                    │
│   - DrugInteractionService (药物相互作用服务)                   │
├─────────────────────────────────────────────────────────────────┤
│ 领域事件:                                                       │
│   - DrugDispensed (药品发放)                                    │
│   - InventoryLowAlert (库存不足预警)                            │

│   - PrescriptionReviewed (处方审核完成)                         │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 业务规则约束

| 规则编号 | 规则名称 | 规则描述 |
|----------|----------|----------|
| BR-PHARM-001 | 药品编码唯一 | 药品编码全局唯一 |
| BR-PHARM-002 | 近效期预警 | 效期≤3个月预警 |
| BR-PHARM-003 | 先进先出原则 | 出库遵循先进先出 |
| BR-PHARM-004 | 麻醉药品五专管理 | 专人、专柜、专锁、专册、专用处方 |
| BR-PHARM-005 | 处方必须审核 | 处方必须经药师审核后方可调配 |
| BR-PHARM-006 | 药物相互作用全覆盖 | 所有药物相互作用需检查 |
| BR-PHARM-007 | 库存不可为负 | 库存数量不能为负数 |
| BR-PHARM-008 | 抗菌药物分级管理 | 非限制/限制/特殊使用分级 |

## 5. 上下文映射

### 5.1 药品上下文与其他上下文的关系

| 上游上下文 | 下游上下文 | 关系模式 | 集成方式 | 说明 |
|------------|------------|----------|----------|------|
| 共享内核 | 药品上下文 | 共享内核 (SK) | 直接引用 | 患者聚合、领域事件定义共享 |
| 系统上下文 | 药品上下文 | 开放主机服务 (OHS) | API调用 | 用户信息、字典数据 |
| 药品上下文 | 财务上下文 | 客户-供应商 (CF) | 领域事件 | 发药触发计费 |
| 门诊上下文 | 药品上下文 | 防腐层 (ACL) | 领域事件+API | 处方审核、发药 |
| 住院上下文 | 药品上下文 | 防腐层 (ACL) | 领域事件+API | 医嘱转处方，库存校验 |
